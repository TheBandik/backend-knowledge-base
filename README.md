## Полезная литература (можно обратиться ко мне за получением)

### Алекс Сюй - System Design. Подготовка к сложному интервью

Книга посвящена подготовке к техническим собеседованиям по системному дизайну.

В ней разобраны типичные задачи (проектирование масштабируемых сервисов, таких как системы хранения, чаты, новостные ленты), подходы к их решению и ключевые архитектурные принципы: масштабирование, отказоустойчивость, кэширование, балансировка нагрузки и работа с данными.

Книга даёт структурированный фреймворк для ответа на вопросы системного дизайна и помогает научиться рассуждать на уровне архитектуры, а не просто перечислять технологии.
